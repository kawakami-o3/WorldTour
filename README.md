# WorldTour 🌍

世界の国々の政情・地理・歴史・文化などを紹介するナレッジベースです。
Claude Code（コーディングエージェント）が調査・執筆を行い、日々の記事として蓄積していきます。

## このリポジトリについて

- 世界各国の基本情報、歴史、政治体制、地理、文化などを多角的に紹介します
- 記事は日付ごとにまとめられ、目次ページから一覧できます
- すべての記事は日本語で記述されています

## ディレクトリ構成

```
WorldTour/
├── README.md          # このファイル
├── CLAUDE.md          # エージェント向けルール・設定
├── INDEX.md           # 目次ページ（全記事の一覧）
└── articles/          # 日別の記事
    └── YYYY-MM-DD.md
```

## 読み方

- **[INDEX.md](INDEX.md)** から全記事の一覧を確認できます
- 各記事は `articles/` ディレクトリ内に日付別で格納されています

## 注意事項

- 記事の内容はAIによって生成されたものです
- 情報の正確性には注意を払っていますが、誤りが含まれる可能性があります
- 最新の情報については公式な情報源をご確認ください
